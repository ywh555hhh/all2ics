# RRULE 高级功能使用指南

all2ics 现在支持完整的 iCalendar RRULE (重复规则) 规范，可以创建各种复杂的重复模式。

## 支持的 RRULE 字段

### 基础字段
- `freq`: 重复频率 (必需)
  - `"YEARLY"` - 每年
  - `"MONTHLY"` - 每月  
  - `"WEEKLY"` - 每周
  - `"DAILY"` - 每日

- `interval`: 重复间隔 (可选)
  - 例如: `"interval": 2` 表示每隔一个周期重复
  - 默认为 1（每个周期重复）

### 结束条件
- `until`: 结束日期 (可选)
  - 格式: `"YYYY-MM-DD"`
  - 例如: `"until": "2025-12-31"`

- `count`: 重复次数 (可选)
  - 例如: `"count": 10` 表示总共重复10次

### 高级过滤
- `byday`: 按星期过滤 (可选)
  - 值: `"MO,TU,WE,TH,FR,SA,SU"`
  - 例如: `"byday": "MO,WE,FR"` 表示周一、三、五

- `bymonth`: 按月份过滤 (可选)
  - 值: `"1,2,...,12"`
  - 例如: `"bymonth": "1,4,7,10"` 表示1月、4月、7月、10月

- `bysetpos`: 按位置过滤 (可选)
  - 值: 正数或负数，支持逗号分隔
  - 例如: `"bysetpos": "1"` 表示第一个
  - 例如: `"bysetpos": "-1"` 表示最后一个
  - 例如: `"bysetpos": "1,-1"` 表示第一个和最后一个

- `wkst`: 一周开始日 (可选)
  - 值: `"MO"`, `"TU"`, `"WE"`, `"TH"`, `"FR"`, `"SA"`, `"SU"`
  - 默认为 `"MO"` (周一)

## 使用示例

### 1. 双周课程
每隔一周重复：
```json
{
  "name": "双周课程",
  "begin": "2025-01-06 14:00:00",
  "end": "2025-01-06 16:00:00",
  "rrule": {
    "freq": "WEEKLY",
    "interval": 2,
    "count": 10
  }
}
```

### 2. 工作日课程
周一到周五：
```json
{
  "name": "工作日课程",
  "begin": "2025-01-06 08:00:00",
  "end": "2025-01-06 09:00:00",
  "rrule": {
    "freq": "WEEKLY",
    "byday": "MO,TU,WE,TH,FR",
    "until": "2025-06-30"
  }
}
```

### 3. 每月第一个周一
月度会议：
```json
{
  "name": "月度会议",
  "begin": "2025-01-06 10:00:00",
  "end": "2025-01-06 12:00:00",
  "rrule": {
    "freq": "MONTHLY",
    "byday": "MO",
    "bysetpos": "1",
    "count": 12
  }
}
```

### 4. 季度课程
每三个月的特定月份：
```json
{
  "name": "季度报告",
  "begin": "2025-01-15 09:00:00",
  "end": "2025-01-15 17:00:00",
  "rrule": {
    "freq": "MONTHLY",
    "interval": 3,
    "bymonth": "1,4,7,10",
    "count": 8
  }
}
```

### 5. 复杂组合：每季度第一个工作日
```json
{
  "name": "季度计划",
  "begin": "2025-01-01 09:00:00",
  "end": "2025-01-01 17:00:00",
  "rrule": {
    "freq": "MONTHLY",
    "interval": 3,
    "bymonth": "1,4,7,10",
    "byday": "MO,TU,WE,TH,FR",
    "bysetpos": "1",
    "wkst": "MO"
  }
}
```

## 使用建议

1. **简单场景优先**: 对于简单的重复（如每周、每月），只使用 `freq` 和 `until`/`count`
2. **渐进式复杂化**: 需要特殊规则时，逐步添加 `interval`、`byday` 等字段
3. **测试验证**: 使用生成的 ICS 文件在日历应用中验证重复规则是否符合预期
4. **字段组合**: 多个字段可以组合使用，创建非常精确的重复模式

## 兼容性说明

- 所有新字段都是可选的，保持向后兼容
- 未指定的字段使用默认值或被忽略
- 生成的 ICS 文件符合 RFC 5545 标准，兼容所有主流日历应用
